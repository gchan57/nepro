This app is Build on focous with SIH 2025 Problem Statement 25062-Implementation of Smart Agriculture for Efficient Cultivation in Hilly Regions..  
Takes data from fields  in realtime and Have Crop Health Analysis with both Cloud and Offline mode. 
Online Mode works via ml model running on colab that is exposed via ngrok
